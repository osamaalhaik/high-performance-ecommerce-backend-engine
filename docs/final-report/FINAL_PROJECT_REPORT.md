# Final Project Report

## Project Title

High-Performance E-Commerce Backend Engine

## Project Objective

The project implements a backend engine for an e-commerce system that handles concurrent users, preserves shared data integrity, and exposes measurable operational evidence through tests, monitoring, and database verification.

## Implemented Functional Scope

- User registration and login
- Admin authentication
- Product listing and product details
- Order creation
- User order retrieval
- Payment persistence
- Admin statistics
- Batch job execution
- Load balancer server selection endpoint

## Non-Functional Scope

- Concurrent access protection
- Resource management
- Asynchronous processing
- Batch processing
- Load distribution
- Redis caching
- Locking and transaction integrity
- Stress testing
- Benchmarking and bottleneck analysis
- AOP-based performance monitoring
